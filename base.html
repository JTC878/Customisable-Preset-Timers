What is this application?

This is a single page application where you can add multiple timers to different times at different days of the week

These timers can be customised from a preset option found in the customise window tab at the top of the page

Customisation includes options such as:

  • Timer name
  • Start time
  • End time or duration
  • Specific days of the week to apply the timer to
  • Browser notification for start time and end time of timer
  • Alarm sound(option to import your own alarm sound)
  • Alarm volume
  • Alarm duration

Countdown Timer Preset

This is the default preset. The timer is used to count down from a specified duration value. An alarm can be set when duration reaches 0.
