What is this application?

This is a single page application where you can add multiple timers to different times at different days of the week

These timers can be customised from a preset option found in the customise window tab at the top of the page

Customisation includes options such as:

  • Timer name
  • Start time
  • End time or duration
  • Specific days of the week to apply the timer to
  • Browser notification for start time and end time of timer
  • Alarm sound(option to import your own alarm sound)
  • Alarm volume
  • Alarm duration

Import and export buttons save/read the state of all timers configured with a start time which can be seen on the active list tab(ListActive)

Countdown Timer Preset

This is the default preset. The timer is used to count down from a specified duration value. An alarm can be set when duration reaches 0.

Select one or more days when the timer will run

Alarm Preset

This is a simple alarm preset. The timer is intended to be a dedicated alarm set at specific times.

Select one or more days for the alarm to be set

Stopwatch Preset

This is a stopwatch timer preset. The stopwatch will increment the duration value and continue running until the user presses the stop button in the running window.

Select one or more days for when the stopwatch will run

Pomodoro Preset

This is a Pomodoro timer preset. The Pomodoro timer has one countdown timer for working/studying and one timer for resting, it will switch when one ends.

Select one or more days for when the pomodoro timer will run
